Spatial VOIP for Unity3d using Mumble and Photon multiplayer..

Tweet    

Tipodean has incorporated the ability to support spatial VOIP within Unity3d. You simple put our code into your Unity3d Photon multiplayer project and you have sptial VOIP. The system can connect to any Mumble/Murmour server allowing you to use a third party to host VOIP or do it yourself. See our demos where we are simply using Photon Cloud as the multiplayer server
